# rubic-core

Pure-Rust core for a 3x3 Rubik's Cube: state model, move engine, input
validation, and solvers (beginner layer-by-layer + optimal). No rendering
dependencies.

Part of the [`rubic`](https://github.com/mingjerli/rubic) project.
